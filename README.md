# FabricPy
## Minecraft modding in Python.

This project is to help people who only know python to make minecraft mods.

Docs:

- `docs/index.md`
- `docs/building.md`
- `docs/mod.md`
- `docs/blocks.md`
- `docs/items.md`
- `docs/recipes.md`
- `docs/textures.md`
- `docs/models.md`
- `docs/events.md`
- `docs/commands.md`
- `docs/mixins.md`
- `docs/decorators.md`
- `docs/context.md`

Project source assets and data now live in:

- `assets/<modid>/...`
- `data/<modid>/...`

Those folders are copied into both generated loader projects during compile.
